使用 cin

与 cout 一样，C++也将输入看作是流入程序的字符流。iostream 文件将 cin 定义为一个表示这种流的对象。输出时，<< 运算符将字符串插入到输出流中；输入时，cin 使用 >> 运算符从输入流中抽取字符。通常，需要在运算符右侧提供一个变量，用于接收抽取的信息。

与 cout 一样，cin 也是一个智能对象它可以通过键盘输入的一系列字符（即输入）转化为接收信息的变量能够接受的形式。在这个例子中，程序员将 carrots 声明为一个整型变量，因此输入被转换成计算机用来存储整数的数字形式。

现在来看 cout、它是一个 ostream 类对象。ostream 类定义（iostream 文件中的另外一个成员）描述了 ostream 对象表示的数据以及可以对它执行的操作，如将数字或字符串插入到输出流中。cin 也是类似的。

注意：类描述了一种数据类型的全部属性（包括可以对它执行的操作），对象是根据这些描述创建的实体。

类买哦输了可对类对象执行的所有操作。要对特定对象执行这些允许的操作，需要给该对象发送一条消息。例如，如果希望 cout 对象显示一个字符串，应该向它发送一条消息，告诉它，“对象！显示这些内容！”。C++提供了两种发送消息的方式：一种是使用类方法（本质上就是稍后介绍的函数调用）；另一种方式就是重新定义运算符，cin 和 cout 使用的就是这种方式。

在给变量取名的时候，一般有如下规则：

  • cout 函数的一项特性——cout.put() 函数，该函数显示一个字符。

  • 成员函数 cout.put() 到底是个什么东西？

    类定义了如何表示和控制数据。成员函数归类所有，描述了操纵类数据的方法。

    只能通过类的特定对象来使用成员函数，使用时，必须用句点将对象名和函数名称(put())连接起来使用。句点被称为成员运算符。cout.put() 的意思是，通过类对象 cout 来使用函数 put()。这里只是一个粗浅的介绍，之后会详细说明。

    要表示一个量是常量，通常使用首字母大写或者全部大写的方法来为产量命名。

    需要注意的是，使用 const 要优于使用 #define。原因如下：

    • 可以指定类型
    • 可以使用 C++的作用域规则将其限制在特定的函数或文件中
    • 可以用于定义更复杂的类型

字符串

字符串是存储在内存的连续字节的一系列字符。C++处理字符串的方式有两种。第一种来自 C 语言，常被称为 C-风格字符串。我们首先介绍它，然后介绍另一种基于 string 库的方法。

C++中可以引用标准头文件 cstring 来提供与字符串相关的函数，这里不再赘述。

字符串输入

  • cin 怎么判断字符串输入完毕呢？

    由于键盘不能输入空字符，所以 cin 需要一些其他的方法来判断字符串的结尾位置。cin 使用空白（空格、制表符和换行符）来确定字符串的结束位置，这意味着 cin 在获取字符数组输入时只能读入一个单词。读取该单词后，cin 将该字符串放到数组中，并自动在结尾添加空字符。

    另外一个问题是，输入字符串可能比目标数组长。这时候仍然使用 cin 很难防止将包含 30 个字符的字符串放到 20 个字符的数组中的情况发生。

幸运的是，istream 中的类（如 cin）提供了一些面向行的类成员函数：getline() 和 get()。这两个函数都读取一行输入，直到到达换行符。然后，getline() 会将换行符丢弃，而 get() 会将换行符保留在输入序列中。

注意：这里指的是“输入序列”，并非把换行符加入字符串。

面向行的输入：getline()

getline() 有两个（暂时认为是两个）参数，第一个参数是用来存储输入行的数组的名称，第二个参数是要读取的字符数。需要注意的是，如果这个参数为 20, 那么这个函数最多读取 19 个字符，因为最后一个字符是空字符。

共用体

共用体（union）是一种数据格式，它能够存储不同的数据类型，但只能同时存储其中的一种类型。也就是说，结构可以同时存储 int、long 和 double，共用体只存存储 int、double 或 long。共用体的句法与结构类似，但含义不同。

显而易见的是，共用体的用途之一是节省存储空间。当前，系统的内存多达数 GB 甚至 数 TB，好像没有必要节省内存，但并非所有的 C++程序都是为这样的系统编写的。C++还可用于嵌入式系统编程，比如烤箱、冰箱和 MP3。对于这些应用程序来说，内存可能十分宝贵。另外，共用体常用语操作系统数据结构或硬件数据结构。
